## 🚀 Quick Start (3 Steps)

### Step 1: Test Locally (5 minutes)

```bash
cd smart-recipe-generator-exp
npm install
npm run dev
```

Open http://localhost:5173 and test all features!

### Step 2: Push to GitHub (10 minutes)

```bash
git init
git add .
git commit -m "Initial commit: Smart Recipe Generator"
git remote add origin https://github.com/askpranay/smart-recipe-generator-exp.git
git branch -M main
git push -u origin main
```

📖 **Detailed guide**: See `GITHUB_SETUP.md`

### Step 3: Deploy to Vercel (5 minutes)

1. Go to [vercel.com](https://vercel.com)
2. Sign in with GitHub
3. Import: `askpranay/smart-recipe-generator-exp`
4. Click "Deploy"
5. Get your live URL!

📖 **Detailed guide**: See `DEPLOYMENT_GUIDE.md`
